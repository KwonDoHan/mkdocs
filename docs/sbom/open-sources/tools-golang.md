# SPDX tools-golang

`tools-golang` 패키지는 Go 프로그램에서 SPDX 파일을 사용하기 편하기 위한 도구들을 모아놓았다.
이미 만들어진 SBOM(SPDX)을 사용하거나, 결과물로 SPDX 출력, 또는 리포팅을 쉽게 출력할 수 있다.
