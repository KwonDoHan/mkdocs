# Delve - Golang Debugger

Delve는 Go 언어용 디버거입니다.
`dlv` 명령어를 사용하여 go 언어를 디버깅할 수 있습니다.

## 설치

```
$ go install github.com/go-delve/delve/cmd/dlv@latest
```

설치가 완료되면 확인해 봅니다.

```
$ dlv version
Delve Debugger
Version: 1.22.1
```

version 정보가 나온다면 설치가 완료된 것입니다.
